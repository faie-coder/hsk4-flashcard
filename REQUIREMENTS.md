# HSK4 Flashcard Application - Requirements Document

## 📋 ภาพรวมโครงการ

แอปพลิเคชัน HSK4 Flashcards เป็นเว็บแอปพลิเคชันสำหรับเรียนรู้คำศัพท์ภาษาจีน HSK ระดับ 4 จำนวน 1,200 คำ พร้อมระบบฝึกหัดและเกมต่างๆ เพื่อช่วยในการจดจำและทบทวนคำศัพท์

---

## 🎯 วัตถุประสงค์หลัก

1. **ให้ผู้เรียนสามารถท่องจำคำศัพท์ HSK4 ได้อย่างมีประสิทธิภาพ**
2. **มีระบบติดตามความก้าวหน้าในการเรียนรู้**
3. **มีแบบฝึกหัดหลากหลายรูปแบบเพื่อเสริมสร้างทักษะ**
4. **รองรับการใช้งานบนอุปกรณ์มือถือและเดสก์ท็อป**

---

## 📊 ข้อมูลคำศัพท์

### โครงสร้างข้อมูล
แต่ละคำศัพท์ประกอบด้วย:
- **n**: หมายเลขลำดับ (1-1200)
- **h**: ตัวอักษรจีน (汉字)
- **p**: Pinyin (การออกเสียง)
- **m**: ความหมายภาษาไทย
- **cat**: หมวดหมู่ (action, noun, adj, adv, other)
- **ex**: ประโยคตัวอย่าง (จีน + Pinyin + ไทย)

### การจัดหมวดหมู่
- **กริยา (action)**: คำกริยาต่างๆ
- **นาม (noun)**: คำนาม
- **คุณศัพท์ (adj)**: คำคุณศัพท์
- **กริยาวิเศษณ์ (adv)**: คำกริยาวิเศษณ์
- **อื่นๆ (other)**: คำเชื่อม, ลักษณนาม, คำอุทาน

### การแบ่งช่วงคำศัพท์
- **ช่วง 1**: คำที่ 1-300
- **ช่วง 2**: คำที่ 301-600
- **ช่วง 3**: คำที่ 601-900
- **ช่วง 4**: คำที่ 901-1200

## 🎨 ส่วนประกอบหลักของ UI

### 1. Sidebar (เมนูด้านซ้าย)

#### 1.1 ส่วนหัว
- ชื่อแอป: "📚 HSK4 Flashcards"
- ปุ่มปิดเมนู (สำหรับมือถือ)

#### 1.2 เมนูแบบฝึกหัด
**สุ่มท่องศัพท์**
- สุ่มท่องศัพท์ 50 คำจากทั้งหมด

**แบบฝึกหัดแบ่งตามช่วง (4 ช่วง)**
แต่ละช่วงมี submenu ประกอบด้วย:
- ✍️ แบบฝึกหัดเขียน
- 📖 อ่าน แล้วตอบคำถาม
- 🔤 เรียงประโยค

#### 1.3 เมนูเครื่องมือ
- ❌ คำที่จำยาก (แสดงจำนวนคำที่ทำเครื่องหมาย)
- ✓ คำที่จำได้ (แสดงจำนวนคำที่ทำเครื่องหมายว่าจำได้)
- 🔍 เปรียบเทียบคำคล้ายกัน
- 🎮 เกมจับคู่ (มีโหมดย่อย: จีน-ไทย/จีน-Pinyin, ระดับ 15/30/50 คำ)
- 🔤 สลับแสดง/ซ่อน Pinyin

#### 1.4 เมนูแบบทดสอบ
- 🎧 โหมดฟังและพิมพ์ (Listening Dictation)
- 📝 แบบทดสอบแบบสุ่ม (3 ระดับความยาก)

#### 1.5 เมนูบทความ
- 📖 ตัวอย่างบทความ (120 บทความ)
- 📝 แบบฝึกหัดตอบคำถาม

### 2. Main Content Area

#### 2.1 Header (ติดด้านบน)
**แถบควบคุม**
- ปุ่ม Filter: เปิด Sidebar กรองตามหมวดหมู่ (หมวดหมู่) จากทางขวา
- ปุ่มคำประสม: เปิด Sidebar กรองคำศัพท์ตามตัวอักษรที่มีในคำ (เช่น 好, 不, 大, 出, 上, 下, ฯลฯ)
- ช่องค้นหา
- แสดงจำนวนคำที่กำลังแสดง
- ระยะห่างระหว่างปุ่ม filter กับข้อความ "จำได้": 24px

**Filter Sidebar (เลื่อนจากขวา)**
- หัวข้อ: ตัวกรองคำศัพท์
- ปุ่มปิด (✕)
- ปุ่ม Reset
- หมวดหมู่ (Radio buttons): ทั้งหมด, กริยาทั่วไป, ร่างกาย, คำคุณศัพท์, อารมณ์, ครอบครัว, อาหาร, คำเชื่อม, เงิน, ธรรมชาติ, สถานที่, สังคม, กีฬา, การเรียน, เทคโนโลยี, เวลา, การเดินทาง, การทำงาน
- แสดงจำนวนคำในแต่ละหมวดหมู่
- z-index: 2001 (สูงกว่า header sticky)
- Overlay: z-index 2000, background rgba(0,0,0,0.5)
- เลื่อนขึ้นด้านบนสุดเมื่อเปิด

**Compound Filter Sidebar (เลื่อนจากขวา)**
- หัวข้อ: คำประสม
- ปุ่มปิด (✕)
- รูปแบบคำประสม 24 รูปแบบ: 好, 不, 大, 出, 上, 下, 打, 发, 人, 心, 学, 工, 以, 时, 来, 成, 关, 经, 感, 自, 高, 一, 方, 家
- กรองคำศัพท์ที่มีตัวอักษรนั้นในคำ
- แสดงตัวอักษรที่เลือกบนปุ่ม เช่น "(好)"
- Overlay: z-index 2000, background rgba(0,0,0,0.5)
- เลื่อนขึ้นด้านบนสุดเมื่อเปิด
- ปิด overlay อัตโนมัติเมื่อเลือกเมนู

**แถบความก้าวหน้า**
- แสดงจำนวนคำที่จำได้
- Progress bar แสดงเปอร์เซ็นต์
- แสดงจำนวนคำที่เรียนรู้แล้ว

#### 2.2 Grid แสดงการ์ดคำศัพท์
**การ์ดด้านหน้า (Front)**
- หมายเลขคำ (มุมบนซ้าย)
- ตัวอักษรจีน (ขนาดใหญ่)
- Pinyin (สามารถซ่อน/แสดงได้)
- ป้ายหมวดหมู่ (สีต่างกันตามประเภท)
- ปุ่มลำดับการเขียน (มุมบนขวา)
- ปุ่มออกเสียง (มุมบนขวา)

**การ์ดด้านหลัง (Back) - เมื่อคลิก**
- ความหมายภาษาไทย
- ประโยคตัวอย่าง (จีน + Pinyin + ไทย)
- คำแนะนำ "คลิกอีกครั้งเพื่อกลับ"

**ปุ่มลบ (มุมบนขวา)**
- แสดงเมื่อ hover
- ใช้สำหรับทำเครื่องหมายคำที่จำยาก

**ปุ่มทำเครื่องหมายจำได้ (✓)**
- แสดงเมื่ออยู่ในหน้าคำที่จำยาก
- ใช้สำหรับทำเครื่องหมายว่าจำได้แล้ว
- เมื่อทำเครื่องหมาย คำนั้นจะถูกลบออกจากรายการคำที่จำยาก

#### 2.3 Pagination
- ปุ่มหน้าก่อนหน้า/ถัดไป
- ปุ่มเลขหน้า
- แสดงข้อมูลหน้าปัจจุบัน/ทั้งหมด

## 🎮 ฟีเจอร์และฟังก์ชันหลัก

### 1. ระบบ Flashcard

#### 1.1 การแสดงผล
- แสดงการ์ดแบบ Grid (responsive)
- รองรับการพลิกการ์ดเพื่อดูความหมาย
- แสดง 50 การ์ดต่อหน้า
- มี pagination สำหรับเปลี่ยนหน้า

#### 1.2 การกรองและค้นหา
- กรองตามหมวดหมู่ (หมวดหมู่) - ผ่าน Filter Sidebar
- กรองตามรูปแบบคำประสม - ผ่าน Compound Dropdown
- ค้นหาจากตัวอักษรจีน, Pinyin, หรือความหมายไทย
- กรองคำที่ทำเครื่องหมายว่าจำยาก
- Reset filters กลับสู่ค่าเริ่มต้น

#### 1.3 การจัดการคำศัพท์
- ทำเครื่องหมายคำที่จำยาก (บันทึกใน localStorage)
- ติดตามความก้าวหน้าการเรียนรู้
- แสดงจำนวนคำที่จำได้

#### 1.4 ฟีเจอร์เสริม
- **ออกเสียง**: ใช้ Web Speech API อ่านออกเสียงภาษาจีน
- **ลำดับการเขียน**: แสดงลำดับการลากเส้นตัวอักษรจีนด้วย HanziWriter
- **สลับ Pinyin**: เปิด/ปิดการแสดง Pinyin
- **ทำเครื่องหมายจำได้**: ใช้ปุ่ม ✓ เพื่อทำเครื่องหมายว่าจำได้แล้ว (บันทึกใน localStorage)

### 2. เปรียบเทียบคำคล้ายกัน (Similar Words Comparison)

#### 2.1 โครงสร้าง
- มีคู่คำคล้ายกันจำนวน 20 คู่
- แต่ละคู่ประกอบด้วย:
  - คำที่ 1: ตัวอักษรจีน, Pinyin, ความหมาย
  - คำที่ 2: ตัวอักษรจีน, Pinyin, ความหมาย
  - ความแตกต่าง: อธิบายความแตกต่างของตัวอักษร
  - ประโยคตัวอย่าง: สำหรับแต่ละคำ (จีน + Pinyin + ไทย)

#### 2.2 การแสดงผล
**รายการคู่คำ (List View)**
- แสดงการ์ดแต่ละคู่พร้อม:
  - ตัวอักษรจีนทั้งสองคำ
  - Pinyin ของทั้งสองคำ
  - ความหมายเปรียบเทียบ
  - ปุ่ม "คลิกเพื่อดูความแตกต่าง"

**รายละเอียดคู่คำ (Detail View)**
- แสดงรายละเอียดของแต่ละคำ:
  - ตัวอักษรจีน (ขนาดใหญ่)
  - Pinyin
  - ความหมาย
  - ประโยคตัวอย่าง (จีน + Pinyin + ไทย)
- แสดงความแตกต่างของตัวอักษร
- ปุ่มกลับไปรายการ

#### 2.3 ตัวอย่างคู่คำ
- 太 (tài) vs 大 (dà)
- 人 (rén) vs 入 (rù)
- 天 (tiān) vs 夫 (fū)
- 日 (rì) vs 目 (mù)
- และอีก 16 คู่

### 3. เกมจับคู่ (Matching Game)

#### 3.1 กลไกเกม
- 2 โหมดการจับคู่:
  - **จีน-ไทย**: จับคู่ตัวอักษรจีนกับความหมายภาษาไทย
  - **จีน-Pinyin**: จับคู่ตัวอักษรจีนกับการออกเสียง Pinyin
- 3 ระดับความยาก:
  - ง่าย: 15 คำ (30 การ์ด)
  - ปานกลาง: 30 คำ (60 การ์ด)
  - ยาก: 50 คำ (100 การ์ด)

#### 3.2 การแสดงผล
- Grid 2 คอลัมน์ (responsive)
- แสดงสถิติ: จำนวนคู่ที่จับได้/ทั้งหมด, เวลาที่ใช้
- Animation เมื่อจับคู่ถูก/ผิด (shake effect)

#### 3.3 กฎการเล่น
- คลิกเลือกการ์ด 2 ใบ
- ถ้าถูก: การ์ดจะเปลี่ยนเป็นสีเขียวและไม่สามารถคลิกได้
- ถ้าผิด: การ์ดจะสั่นและกลับคืนสู่สถานะเดิม
- จบเกมเมื่อจับคู่ครบทั้งหมด
- แสดงเวลาที่ใช้เมื่อจบเกม

### 4. โหมดฟังและพิมพ์ (Listening Dictation)

#### 4.1 โครงสร้าง
- สุ่มคำศัพท์ 10 คำจากทั้งหมด 1,200 คำ
- ใช้ Text-to-Speech (Web Speech API) เล่นเสียงภาษาจีน

#### 4.2 รูปแบบการฝึก
- กดปุ่ม "เล่นเสียง" เพื่อฟังคำศัพท์
- พิมพ์ตัวอักษรจีนที่ได้ยินลงในช่อง input
- กด "ตรวจคำตอบ" หรือปุ่ม Enter

#### 4.3 การให้คะแนน
- ตรวจสอบความถูกต้องอัตโนมัติ
- แสดงคำตอบที่ถูกต้อง (ตัวอักษรจีน + Pinyin + ความหมาย)
- นับคะแนนแบบ real-time
- มีปุ่ม "ข้าม" เพื่อดูคำตอบและไปข้อถัดไป

### 5. แบบทดสอบแบบสุ่ม (Random Quiz)

#### 5.1 โครงสร้าง
- สุ่มคำถามจากคำศัพท์ทั้งหมด 1,200 คำ
- 10 คำถามต่อรอบ
- สามารถเปลี่ยนระดับความยากระหว่างทำแบบทดสอบได้

#### 5.2 ระดับความยาก
- **ง่าย (Easy)**: แสดงตัวอักษรจีน + Pinyin, เลือกความหมายจาก 4 ตัวเลือก
- **ปานกลาง (Medium)**: แสดงตัวอักษรจีน + Pinyin + ความหมาย, เติมตัวอักษรตัวแรกที่หายไป
- **ยาก (Hard)**: แสดง Pinyin + ความหมาย, เขียนตัวอักษรจีนทั้งคำ

#### 5.3 การให้คะแนน
- ตรวจสอบคำตอบทันที
- แสดง feedback ถูก/ผิด พร้อมคำตอบที่ถูกต้อง
- ถ้าตอบถูก: ไปข้อถัดไปอัตโนมัติหลัง 0.8 วินาที
- ถ้าตอบผิด: ต้องกดปุ่มเพื่อไปข้อถัดไป
- สรุปคะแนนรวม + กราฟความก้าวหน้าเมื่อจบ

### 6. แบบฝึกหัดเขียน (Writing Practice)

#### 6.1 โครงสร้าง
- เลือกช่วงคำศัพท์ (1-300, 301-600, 601-900, 901-1200)
- สุ่มคำจากช่วงที่เลือก จำนวน 10 คำ

#### 6.2 รูปแบบการฝึก
- แสดง Pinyin และความหมายไทย
- ผู้เรียนเขียนตัวอักษรจีนบน canvas
- มีปุ่มล้างและตรวจสอบ
- แสดงคำถัดไปอัตโนมัติหลังตรวจสอบ
- **ปุ่มแสดง/ซ่อนตัวอักษร**: สลับการแสดง outline ของตัวอักษรบน canvas (ใช้ HanziWriter)

#### 6.3 การตรวจสอบ
- ใช้ HanziWriter ในการตรวจสอบ (ถ้ามี)
- แสดง feedback ว่าถูกหรือผิด
- นับคะแนน

### 7. แบบฝึกหัดอ่านและตอบคำถาม (Reading Quiz)

#### 7.1 โครงสร้าง
- เลือกช่วงคำศัพท์
- สุ่มคำถาม 15 ข้อ
- แต่ละข้อมี 4 ตัวเลือก

#### 7.2 ประเภทคำถาม
- อ่านประโยคจีน แล้วเลือกความหมายที่ถูกต้อง
- อ่านความหมายไทย แล้วเลือกคำจีนที่ถูกต้อง
- เติมคำในช่องว่าง

#### 7.3 การให้คะแนน
- แสดงผลทันทีว่าถูกหรือผิด
- แสดงคำตอบที่ถูกต้อง
- สรุปคะแนนรวมเมื่อจบ

### 8. แบบฝึกหัดเรียงประโยค (Sentence Ordering)

#### 8.1 โครงสร้าง
- เลือกช่วงคำศัพท์
- สุ่มประโยค 15 ประโยค

#### 8.2 รูปแบบการฝึก
- แสดงความหมายไทยของประโยค
- แสดง Pinyin (สามารถสลับเปิด/ปิดได้ด้วยปุ่ม "🔤 แสดง/ซ่อน Pinyin")
- แสดงคำศัพท์แบบสุ่มเรียง
- ผู้เรียนคลิกเรียงคำตามลำดับที่ถูกต้อง

#### 8.3 การตรวจสอบ
- ตรวจสอบลำดับคำ
- แสดง feedback ถูก/ผิด
- แสดงคำตอบที่ถูกต้อง
- นับคะแนน

### 9. บทความตัวอย่าง (Sample Articles)

#### 9.1 โครงสร้าง
- มีบทความทั้งหมด 120 บทความ
- แบ่งตามช่วงคำศัพท์ (1-300, 301-600, 601-900, 901-1200)

#### 9.2 เนื้อหาบทความ
- ชื่อบทความ
- เนื้อหาแบ่งเป็นพารากราฟ
- แต่ละพารากราฟมี:
  - ข้อความภาษาจีน
  - Pinyin
  - คำแปลภาษาไทย

#### 9.3 ฟีเจอร์
- กรองบทความตามช่วงคำศัพท์
- คลิกคำศัพท์ในบทความเพื่อดูความหมาย
- สลับแสดง/ซ่อน Pinyin
- สลับแสดง/ซ่อนคำแปลไทย

### 10. แบบฝึกหัดตอบคำถามจากบทความ (Article Quiz)

#### 10.1 โครงสร้าง
- เลือกช่วงคำศัพท์
- แสดงบทความพร้อมคำถาม
- แต่ละบทความมี 1-3 คำถาม

#### 10.2 รูปแบบ
- แสดงเนื้อหาบทความ (จีน + Pinyin + ไทย)
- แสดงคำถามเป็นภาษาไทย
- มี 4 ตัวเลือก
- สามารถสลับแสดง/ซ่อนคำแปลไทยได้

#### 10.3 การให้คะแนน
- แสดงผลทันทีว่าถูกหรือผิด
- แสดงคำตอบที่ถูกต้อง
- สรุปคะแนนรวมเมื่อจบ

### 11. ลำดับการเขียนตัวอักษร (Stroke Order)

#### 11.1 เทคโนโลยี
- ใช้ HanziWriter library
- แสดงลำดับการลากเส้นแบบ animated

#### 11.2 ฟีเจอร์
- แสดงการเขียนทีละตัวอักษร
- แสดง radical (ส่วนประกอบหลัก) เป็นสีน้ำเงิน
- แสดงขีดทั่วไปเป็นสีดำ
- มีปุ่มเล่นซ้ำ

#### 11.3 การแสดงผล
- แสดงตัวอักษรทั้งหมมในคำ
- แสดงลำดับการเขียนทีละตัว
- มี legend อธิบายสี

## 🎯 User Flows

### 1. การเรียนรู้คำศัพท์พื้นฐาน
```
1. ผู้ใช้เปิดแอป
2. เห็นการ์ดคำศัพท์ 50 คำแรก
3. คลิกการ์ดเพื่อดูความหมาย
4. คลิกปุ่มออกเสียงเพื่อฟังการออกเสียง
5. คลิกปุ่มลำดับการเขียนเพื่อดูวิธีเขียน
6. ทำเครื่องหมายคำที่จำยากด้วยปุ่มลบ
7. เปลี่ยนหน้าเพื่อดูคำถัดไป
```

### 2. การกรองและค้นหา
```
1. คลิกปุ่มหมวดหมู่ (เช่น "กริยา")
2. ระบบกรองแสดงเฉพาะคำในหมวดหมู่นั้น
3. หรือพิมพ์คำค้นหาในช่องค้นหา
4. ระบบกรองแสดงเฉพาะคำที่ตรงกับคำค้นหา
```

### 3. การเล่นเกมจับคู่
```
1. คลิก "เกมจับคู่" ในเมนู
2. ระบบสุ่มคำ 30 คำ
3. ผู้เล่นคลิกเลือกการ์ด 2 ใบ
4. ถ้าถูก: การ์ดเปลี่ยนสีเขียว
5. ถ้าผิด: การ์ดสั่นและกลับคืน
6. เล่นต่อจนจับคู่ครบ
7. แสดงเวลาที่ใช้
```

### 4. การทำแบบฝึกหัดเขียน
```
1. คลิก "แบบฝึกหัดเขียน" ในช่วงที่ต้องการ
2. ระบบแสดง Pinyin และความหมาย
3. ผู้เรียนเขียนตัวอักษรจีนบน canvas
4. คลิกปุ่ม "ตรวจสอบ"
5. ระบบแสดง feedback
6. ไปยังคำถัดไป
7. จบเมื่อครบ 10 คำ
```

### 5. การอ่านบทความ
```
1. คลิก "ตัวอย่างบทความ" ในเมนู
2. เลือกช่วงคำศัพท์
3. เลือกบทความที่ต้องการอ่าน
4. อ่านบทความพร้อม Pinyin และคำแปล
5. คลิกคำศัพท์เพื่อดูความหมาย
6. สลับแสดง/ซ่อน Pinyin หรือคำแปลตามต้องการ
```

## ⚙️ ฟังก์ชัน JavaScript หลัก

### 1. Core Functions

#### `render()`
- แสดงการ์ดคำศัพท์ตามเงื่อนไขการกรอง
- จัดการ pagination
- อัปเดต UI

#### `setTopic(topic, el)`
- กรองคำศัพท์ตามหมวดหมู่ (topic)
- อัปเดตสถานะปุ่ม active
- ปิด Filter Sidebar อัตโนมัติ

#### `setCompound(key, el)`
- กรองคำศัพท์ตามรูปแบบคำประสม
- อัปเดตข้อความบนปุ่ม (แสดงตัวอักษร เช่น "(好)")
- ปิด Compound Sidebar และ Overlay อัตโนมัติ

#### `toggleFilterSidebar()`
- เปิด/ปิด Filter Sidebar จากทางขวา
- แสดง overlay พร้อม z-index 2000
- เลื่อนขึ้นด้านบนสุดเมื่อเปิด

#### `toggleCompoundDropdown()`
- เปิด/ปิด Compound Filter Sidebar จากทางขวา
- แสดง overlay พร้อม z-index 2000
- เลื่อนขึ้นด้านบนสุดเมื่อเปิด

#### `updateFilterCounts()`
- อัปเดตจำนวนคำในแต่ละหมวดหมู่
- แสดงใน Filter Sidebar

#### `doSearch(query)`
- ค้นหาคำศัพท์จากตัวอักษรจีน, Pinyin, หรือความหมาย
- อัปเดตการแสดงผล

#### `flipCard(index)`
- พลิกการ์ดเพื่อแสดงความหมาย/ประโยคตัวอย่าง

#### `removeDifficult(index)`
- เพิ่ม/ลบคำออกจากรายการคำที่จำยาก
- บันทึกใน localStorage

#### `toggleLearned(n)`
- เพิ่ม/ลบคำออกจากรายการคำที่จำได้
- บันทึกใน localStorage
- ถ้าทำเครื่องหมายว่าจำได้ และคำนั้นอยู่ในรายการคำที่จำยาก จะลบคำนั้นออกจากรายการคำที่จำยาก

### 2. Practice Functions

#### `startPractice(range)`
- เริ่มโหมดฝึกท่องศัพท์
- สุ่มคำ 50 คำจากช่วงที่เลือก

#### `startListeningDictation()`
- เริ่มโหมดฟังและพิมพ์
- สุ่มคำ 10 คำ
- ใช้ Text-to-Speech เล่นเสียง

#### `startRandomQuiz()`
- เริ่มแบบทดสอบแบบสุ่ม
- สุ่มคำถาม 10 ข้อ
- รองรับ 3 ระดับความยาก

#### `setQuizMode(mode)`
- เปลี่ยนระดับความยากระหว่างทำแบบทดสอบ
- รีเซ็ตคะแนนและเริ่มใหม่

#### `checkAndNextQuiz()`
- ตรวจคำตอบและไปข้อถัดไปในปุ่มเดียว
- ถ้าถูก: ไปอัตโนมัติ
- ถ้าผิด: ต้องกดอีกครั้งเพื่อไปต่อ

#### `startRangePractice(start, end)` (ถ้ามี)
- เริ่มแบบฝึกหัดเขียน
- สุ่มคำ 10 คำจากช่วงที่กำหนด

#### `toggleWritingOutline()`
- สลับการแสดง/ซ่อน outline ของตัวอักษรในแบบฝึกหัดเขียน
- ใช้ HanziWriter.showOutline() และ HanziWriter.hideOutline()
- อัปเดตข้อความบนปุ่ม

#### `startReadingQuiz(start, end)`
- เริ่มแบบฝึกหัดอ่านและตอบคำถาม
- สุ่มคำถาม 15 ข้อ

#### `startSentenceOrdering(start, end)`
- เริ่มแบบฝึกหัดเรียงประโยค
- สุ่มประโยค 15 ประโยค

#### `toggleOrderingPinyin()`
- สลับการแสดง/ซ่อน Pinyin ในแบบฝึกหัดเรียงประโยค
- อัปเดตข้อความบนปุ่ม

### 3. Game Functions

#### `startMatchingGame(mode, pairCount)`
- เริ่มเกมจับคู่
- รับพารามิเตอร์: mode ('chinese-thai' หรือ 'chinese-pinyin'), pairCount (15, 30, หรือ 50)
- สร้างการ์ดตามจำนวนคู่ที่กำหนด

#### `selectGameCard(index)`
- จัดการการเลือกการ์ดในเกม
- ตรวจสอบการจับคู่

### 4. Similar Words Functions

#### `showSimilarWords()`
- แสดงรายการคู่คำคล้ายกันทั้งหมด (20 คู่)
- แสดงในรูปแบบการ์ดพร้อมตัวอักษรจีน, Pinyin, และความหมาย

#### `showSimilarDetail(index)`
- แสดงรายละเอียดของคู่คำคล้ายกันที่เลือก
- แสดงความแตกต่างของตัวอักษร
- แสดงประโยคตัวอย่างสำหรับแต่ละคำ

### 5. Article Functions

#### `showSampleArticles()`
- แสดงรายการบทความ
- กรองตามช่วงคำศัพท์

#### `showArticle(articleId)`
- แสดงเนื้อหาบทความ
- จัดการการแสดง Pinyin และคำแปล

#### `showQuiz(range)`
- แสดงแบบฝึกหัดตอบคำถามจากบทความ
- กรองคำถามตามช่วงที่เลือก

### 6. Utility Functions

#### `speakChinese(text)`
- ใช้ Web Speech API อ่านออกเสียงภาษาจีน
- ตั้งค่า rate, pitch, และเลือก voice

#### `showStrokeOrder(wordIndex)`
- แสดงลำดับการเขียนตัวอักษรด้วย HanziWriter
- จัดการ animation

#### `togglePinyin()`
- สลับการแสดง/ซ่อน Pinyin
- บันทึกการตั้งค่าใน localStorage

#### `toggleSidebar()`
- เปิด/ปิด sidebar (สำหรับมือถือ)
- จัดการ overlay

## 🔒 ข้อจำกัดและข้อควรระวัง

### 1. Browser Compatibility
- ต้องการเบราว์เซอร์ที่รองรับ:
  - ES6+ JavaScript
  - CSS Grid และ Flexbox
  - localStorage API
  - Web Speech API (สำหรับการออกเสียง)

### 2. Performance
- ไฟล์ HTML มีขนาดใหญ่ (360KB) เนื่องจากมีข้อมูลคำศัพท์ 1,200 คำ
- การ render การ์ดจำนวนมากอาจใช้ memory สูง
- จำกัดการแสดงผลที่ 50 การ์ดต่อหน้า

### 3. Data Persistence
- ข้อมูลเก็บใน localStorage อาจหายหากผู้ใช้ล้างข้อมูลเบราว์เซอร์
- ไม่มีการ sync ข้อมูลระหว่างอุปกรณ์

### 4. Offline Support
- แอปทำงานได้แบบ offline หลังจากโหลดครั้งแรก
- ยกเว้น HanziWriter library ที่ต้องโหลดจาก CDN
